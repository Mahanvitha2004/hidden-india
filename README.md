A responsive tourism website showcasing India's lesser-known travel destinations. The website was developed as part of a Full Stack Development internship using WordPress, Elementor, HTML, CSS, and JavaScript. It focuses on providing an engaging and user-friendly experience for discovering hidden tourist destinations across India.
Features
Responsive design for desktop, tablet, and mobile devices
Interactive India map with state-wise navigation
Home, About, Discover, Stories, and Contact pages
Clean and modern user interface
Responsive layouts built using Elementor
WordPress theme and plugin customization
